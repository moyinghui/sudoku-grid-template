Game Rules

Sudoku is a logic-based puzzle game played on a grid. The goal is simple: fill every empty cell with a number so that:

  • Each row contains all digits exactly once.
  • Each column contains all digits exactly once.
  • Each sub-grid (region) also contains all digits exactly once.
